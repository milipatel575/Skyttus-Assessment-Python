# Bank Management System

## Project Description

A simple Bank Management System developed using Python and MySQL.

## Features

- Create Bank Account
- Savings and Current Account Types
- Initial Deposit
- Deposit Money
- Withdraw Money
- Check Balance
- Search Account
- Update Customer Name
- Delete Account
- View Transaction History
- View All Accounts
- Bank Summary Report

## Technologies Used

- Python
- MySQL
- mysql-connector-python
- VS Code

## Database

bank_System_db

Table:

accounts

## How To Run

1. Execute bank.sql
2. Install mysql-connector-python
3. Update MySQL password in bank.py
4. Run:

python bank.py
